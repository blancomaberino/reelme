# apps/mobile

Expo React Native app (TypeScript) — scaffolded in **T-004**.

The Reelmap mobile client: share-sheet ingestion (`expo-share-intent`), maps (`react-native-maps`), and the diner/influencer/restaurant flows. Builds run through EAS.

> **A custom EAS dev client is required** — Expo Go will not work because the share intent and native maps depend on config plugins.
